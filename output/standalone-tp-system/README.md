# standalone-tp-system

## Framework
CFX_DEFAULT

## Installation
1. Copy the folder 'standalone-tp-system' into your server resources folder.
2. Add 'ensure standalone-tp-system' to your server.cfg.
3. Configure your teleport locations in config.lua using vector4 (the 4th value is the player heading/rotation).
4. If you enable Ace Permissions, add lines like `add_ace identifier.license:xxx "teleport.fib_elevator" allow` to your permissions.cfg.

## Files
- `client.lua` - Client-side logic
- `server.lua` - Server-side logic
- `config.lua` - Configuration options
- `fxmanifest.lua` - Resource manifest
